# Text

Use different type of text to display different kind of info.

```html
<n-text type="info">Info</n-text>&nbsp;
<n-text type="success">Success</n-text>&nbsp;
<n-text type="warning">Warning</n-text>&nbsp;
<n-text type="error">Error</n-text>&nbsp; <n-text strong>Strong</n-text>&nbsp;
<n-text italic>Italic</n-text>&nbsp; <n-text underline>Underline</n-text>&nbsp;
<n-text delete>Delete</n-text>&nbsp; <n-text code>Code</n-text>&nbsp;
<n-text code delete>Code</n-text>&nbsp;
<n-text depth="1">Primary Depth</n-text>&nbsp;
<n-text depth="2">Secondary Depth</n-text>&nbsp;
<n-text depth="3">Tertiary Depth</n-text>&nbsp;
<n-text tag="div">As Div</n-text>
```
