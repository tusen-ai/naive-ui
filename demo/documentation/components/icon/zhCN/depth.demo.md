# 深度

为了搭配不同级的文字颜色，图标提供 `depth` 选项。

```html
<n-icon size="40" :depth="1">
  <cash-outline />
</n-icon>
<n-icon size="40" :depth="2">
  <cash-outline />
</n-icon>
<n-icon size="40" :depth="3">
  <cash-outline />
</n-icon>
<n-icon size="40" :depth="4">
  <cash-outline />
</n-icon>
<n-icon size="40" :depth="5">
  <cash-outline />
</n-icon>
```

```js
import { CashOutline } from '@vicons/ionicons-v5'

export default {
  components: {
    CashOutline
  }
}
```
