# 元素 Element

Element 上面有很多 Naive UI 提供的主题变量。

## 演示

```demo
basic.vue
```

## API

### Element Props

| 名称 | 类型     | 默认值  | 说明                           |
| ---- | -------- | ------- | ------------------------------ |
| tag  | `string` | `'div'` | `n-element` 需要渲染为什么元素 |

### Element Slots

| 名称    | 参数 | 说明     |
| ------- | ---- | -------- |
| default | `()` | 元素内容 |
