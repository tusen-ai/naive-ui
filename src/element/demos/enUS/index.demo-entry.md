# Element

Element has many theme variables provided by Naive UI.

## Demos

```demo
basic.vue
```

## API

### Element Props

| Name | Type     | Default | Description                                |
| ---- | -------- | ------- | ------------------------------------------ |
| tag  | `string` | `'div'` | The tag `n-element` should be rendered as. |

### Element Slots

| Name    | Parameters | Description                 |
| ------- | ---------- | --------------------------- |
| default | `()`       | The content of the element. |
