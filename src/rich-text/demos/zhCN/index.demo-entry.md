# 富文本 RichText

文本展示，自动将html标签编译成代码展示。

自 `NEXT_VERSION` 开始提供。

## 演示

```demo
basic.vue
```

## API

### RichText Props

| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| content | `string` | `''` | 富文本内容 | NEXT_VERSION |
| DompurifyConfig | `DompurifyConfig` | `{}` | 参考 `dompurify` 的 `Config` | NEXT_VERSION |
