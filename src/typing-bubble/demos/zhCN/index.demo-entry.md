## API

### TypingBubble Props

| 名称    | 类型     | 默认值      | 说明                   | 版本         |
| ------- | -------- | ----------- | ---------------------- | ------------ |
| content | `string` | `undefined` | 气泡内要展示的文本内容 | NEXT_VERSION |
| isTyping | `Boolean` | `false` | 气泡内要展示的文本内容 | NEXT_VERSION |
