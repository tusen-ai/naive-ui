# Time

Time provide some basic formation for time.

## Demos

```demo
basic
type
format
relative
unix
```

## Props

| Name   | Type                                 | Default      | Description |
| ------ | ------------------------------------ | ------------ | ----------- |
| format | `string`                             | `undefined`  |             |
| time   | `number \| Date`                     | `Date.now()` |             |
| to     | `number \| Date`                     | `Date.now()` |             |
| type   | `'relative' \| 'date' \| 'datetime'` | `'datetime'` |             |
| unix   | `boolean`                            | `false`      |             |
