## API

### List Props

| 名称         | 类型      | 默认值  | 说明                     | 版本         |
| ------------ | --------- | ------- | ------------------------ | ------------ |
| bordered     | `boolean` | `false` | 是否显示边框             |              |
| clickable    | `boolean` | `false` | 列表项是否有可点击样式   | NEXT_VERSION |
| hoverable    | `boolean` | `false` | 列表项是否有悬浮样式     | NEXT_VERSION |
| show-divider | `boolean` | `true`  | 是否显示标项之间的分割线 | NEXT_VERSION |

### List Slots

| 名称    | 参数 | 说明           |
| ------- | ---- | -------------- |
| default | `()` | 列表的内容     |
| footer  | `()` | 列表底部的内容 |
| header  | `()` | 列表头部的内容 |

### ListItem Slots

| 名称    | 参数 | 说明             |
| ------- | ---- | ---------------- |
| default | `()` | 列表项的内容     |
| prefix  | `()` | 列表项的首部内容 |
| suffix  | `()` | 列表项的尾部内容 |
